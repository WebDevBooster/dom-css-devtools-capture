DOM + CSS Capture

Start before you edit in Elements. Export creates a ZIP containing the current live DOM, accessible CSS snapshots, actual accessible CSS rule changes and a minimal mutation/change log. You can also export only the mutation log JSON.

Status

Not started.

What this captures

  • The current rendered DOM as current-dom.html.
  • A before/after DOM snapshot when capture was started before editing.
  • DOM changes made in the Elements panel, recorded through a MutationObserver.
  • Accessible CSS rules from document.styleSheets, including before/after rule diffs when the browser exposes them.
  • A minimal mutation-log.json containing the edited URL and actual changes.

Known limits

  • This is a live-page snapshot, not a clean patch against your developer's source files.
  • Cross-origin stylesheets may be listed but their rules may be inaccessible.
  • CSS hidden behind browser/security restrictions cannot be exported by page JavaScript.
  • Start capture before editing if you want a useful mutation log.
